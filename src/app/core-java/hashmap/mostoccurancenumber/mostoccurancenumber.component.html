Most Occurrence Number


Program to find most occurrence of number in array using Hashmap.

Example :

If we have an array int array[] = {{ '{' }}10,20,30,20,20{{ '}' }} and 
we need to print the number which is repeated more number of times in an array. 
In the array we can see the number 20 is repeated thrice. 
So, we can print the number 20 is the most occurrence number in the array.

Sample 1 :
Array = {{ '{' }}1,2,3,2,2{{ '}' }}
output : 
Most Occurrence number is 2

Sample 2 :
Array = {{ '{' }}11,23,32,32,23{{ '}' }}
output : 
Most Occurrence number is 23
Most Occurrence number is 32
Example - 1
Input

array = {{ '{' }}1,2,3,2,2{{ '}' }}

Output

Most occurence number is 2

Example - 2
Input

array = {{ '{' }}11,5,12,12,5,5,5{{ '}' }}

Output

Most occurence number is 5
