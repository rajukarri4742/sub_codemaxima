Unique Numbers


Program to find unique numbers in array using Hashmap.

Example :

If we have an array int array[] = {{ '{' }}10,20,30,20,20{{ '}' }} and 
we need to print the number which is repeated only once in an array. 
In the array we can see the numbers 10 and 30 are repeated only one time. 
So, we can print the numbers 10 and 30 are the unique numbers in the array.

Sample 1 :
Array = {{ '{' }}1,2,3,2,5{{ '}' }}
output : 
The unique numbers in the array are 1 3 5.

Sample 2 :
Array = {{ '{' }}11,23,32,23,23{{ '}' }}
output : 
The unique numbers in the array are 11 32.
Example - 1
Input

Array = {{ '{' }}1,2,3,2,5{{ '}' }}

Output

The unique numbers in the array are 1 3 5

Example - 2
Input

Array = {{ '{' }}11,23,32,23,23{{ '}' }}

Output

The unique numbers in the array are 11 32

Example - 3
Input

Array = {{ '{' }}22,23,85,23,23{{ '}' }}

Output

The unique numbers in the array are 22 85
