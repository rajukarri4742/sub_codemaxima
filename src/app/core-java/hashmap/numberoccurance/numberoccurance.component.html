Each Number Occurrence


Program to find number of occurrence of each number in the array using Hashmap.

Example :

If we have an array int array[] = {{ '{' }}1,2,3,2,2{{ '}' }} and the first number is 1 then the number of occurrence of number is 1 and 
and the next number is 2 so the number of occurrence of number 2 is 3.
In the same way, we need to print the number of occurrence of every element in an array.

Sample 1 :
Array = {{ '{' }}1,2,3,2,2{{ '}' }}
output : 
The number of occurrence of number 1 is 1
The number of occurrence of number 2 is 3
The number of occurrence of number 3 is 1

Sample 2 :
Array = {{ '{' }}11,23,32,32,23{{ '}' }}
output : 
The number of occurrence of number 11 is 1
The number of occurrence of number 23 is 2
The number of occurrence of number 32 is 2
Example - 1
Input

array = {{ '{' }}1,2,3,2,2{{ '}' }}

Output

The number of occurence of a number 1 is 1
The number of occurence of a number 2 is 3
The number of occurence of a number 3 is 1

Example - 2
Input

array = {{ '{' }}11,5,12,12,5,5,5{{ '}' }}

Output

The number of occurence of a number 11 is 1
The number of occurence of a number 12 is 2
The number of occurence of a number 5 is 4

Example - 3
Input

array = {{ '{' }}22,22,85,85,85{{ '}' }}

Output

The number of occurence of a number 22 is 2
The number of occurence of a number 85 is 3
