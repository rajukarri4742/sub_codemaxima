Most Occurrence String


  Program to find most occurrence string in string array using Hashmap.

  Example :
  If we have an array string[] = {{ '{' }} “java”,”test”,”java”{{ '}' }} and 
  we need to print the string which is repeated more number of times in a string array. 
  In the string array we can see the string “java” is repeated twice. 
  So, we can print the string "java” is the most occurrence string in the string array.

  Sample 1 :
  String[] = {{ '{' }}“psycho”,”coding”,”maxima”,”psycho”{{ '}' }}
  output : 
  Most Occurrence string in the array is psycho

  Sample 2 :
  String[] = {{ '{' }}“january”,”november”,”october”,”august”,“january”,“november”{{ '}' }}
  output : 
  Most Occurrence string in the string array is january 
  Most Occurrence string in the array is november

Input

string[] = {{ '{' }}“psycho”,”coding”,”maxima”,”psycho”{{ '}' }}
