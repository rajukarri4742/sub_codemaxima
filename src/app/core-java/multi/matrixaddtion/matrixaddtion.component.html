Matrix Addition

We need to find addition of two matrices.If both the matrices are in the same order then only addition of two matrices is possible.
Example: 2X2 matrix
matrix A = 1 1 4 2
matrix B = 6 1 2 5
matrix C = 1+6 1+1 4+2 2+5 = 7 2 6 7

Example - 1
Input

Enter number of rows

2

Enter number of columns

2

Enter elements of matrix A

1 2 3 4

Enter elements of matrix B

5 6 7 8

Output

Addition of two matrices is

6 8 10 12

Input

Enter number of rows
