Matrix Multiplication

We need to add each and every element present in the both diagonals(primary and secondary diagonals) but we should count the element only once.
Example: 3X3 matrix
matrix A = 11 1 8 2 6 3 8 2 9
sum of primary diagonal = 11 + 6 + 9 = 26
sum of secondary diagonal = 8 + 8 = 16 (We cannot include 6 as we should count the element only once.)
Sum = 26 + 16 = 24

Example - 1
Input

Enter number of rows

3

Enter number of columns

3

Enter elements of matrix

1 2 3 4 5 6 7 8 8

Output

Diagonal sum of matrix is

23

Example - 2
Input

Enter number of rows

2

Enter number of columns

2

Enter elements of matrix

Output

Diagonal sum of matrix is

24

Example - 3
