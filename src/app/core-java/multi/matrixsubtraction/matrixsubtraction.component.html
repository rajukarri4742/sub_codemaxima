Matrix Subtraction

We need to find subtraction of two matrices.If both the matrices are in the same order then only subtraction of two matrices is possible.
Example: 2X2 matrix
matrix A = 11 1 4 12
matrix B = 6 1 2 5
matrix C = 11-6 1-1 4-2 12-5 = 5 0 2 7

Example - 1
Input

Enter number of rows

2

Enter number of columns

2

Enter elements of matrix A

5 6 7 8

Enter elements of matrix B

1 2 3 4

Output

Subtraction of two matrices is

4 4 4 4
