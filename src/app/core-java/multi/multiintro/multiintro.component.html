Multi Dimensional Array Introduction


Multi-dimensional arrays:

Multi-dimensional arrays include 2D arrays and 3D arrays i.e. Two dimensional arrays and three-dimensional arrays.

2D Arrays:

Two Dimensional Arrays are also called as Double dimensional arrays or Tabular arrays or Rectangular Arrays.
It is an array of one-dimensional arrays.
It represents the elements in the form of a matrix.
It will store the elements in the form of rows and columns.

Declaration:

datatype [ ][ ] arrayname = new datatype[rowsize ][columnsize ]
int [ ][ ] array = new int [2][2]; // rows = 2, columns = 2
array [0][0] = 1
array [0][1] = 5
array [1][0] = 2
array [1][1] = 6
