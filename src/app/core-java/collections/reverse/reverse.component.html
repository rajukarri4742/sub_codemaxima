Reverse of an element in the ArrayList

We need to print the elements of an arraylist in the reverse order that is, the last element should be displayed first, followed by second last element and so on.

Example - 1
Input

Enter size of an arraylist

5

Enter elements of an arraylist

23

34

12

66

67

Output

Reverse of an arraylist is

67

66

12

34

23

Example - 2
Input

Enter size of an arraylist

4

Enter elements of an arraylist

2

5

9

1

Output

Reverse of an arraylist is

1

9

5

2

Example - 3
Input

Enter size of an arraylist

3

Enter elements of an arraylist

10

28

26

Output

Reverse of an arraylist is

26

28

10
