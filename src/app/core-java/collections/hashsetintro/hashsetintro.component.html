Hashset Introduction


Hashset is implemented based on the hashtable datastructure.

Hashset does not allow duplicate values. 
If we try to add the object then the object will not add into the hashset and the method “add” will return false.

We can insert null values into the hashset.

Objects are inserted into the hashset based on the hashcode of the objects.

We cannot iterate for loop through index in hashset.

Creating Hashset:

HashSet<String> set = new HashSet ();
Here set can store strings only.

HashSet<Integer> set = new HashSet ();
Here set can store integers only.
