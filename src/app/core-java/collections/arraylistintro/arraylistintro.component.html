ArrayList Introduction


Array List is a resizable array or dynamic array. 

As we know that the size of an array is fixed, if we declare an array of size 10 then we cannot store more than 10 elements in the array. 

But in the case of array list the size of an array list is not fixed. It is dynamic. The size of an array list can be changed according to the elements inside it.

We cannot change or modify an array means if we want to add elements to an array or want to delete elements from an array you need to create a new array. But in the case of an array list, we can add or remove elements from an array list whenever we want.

Creating an Array List:

ArrayList<Integer> integers = new ArrayList<> ();
Here, integers are the object which can store integer values.
ArrayList<String> strings = new ArrayList<> ();
Here, strings are the object which can store Strings.
