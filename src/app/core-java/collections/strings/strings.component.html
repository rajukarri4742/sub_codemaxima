Find String in an ArrayList

We need to find if a string exists in an arraylist or not. we need to compare every string in an arraylist with the string which we want to find. if we get matched with the string in an array then we can print like string found otherwise print string not found.

Example - 1
Input

Enter size of an arraylist

5

Enter elements of an arraylist

coding

maxima

student

learning

success

Enter a string

success

Output

string success is found in the arraylist

Example - 2
Input

Enter size of an arraylist

3

Enter elements of an arraylist

january

october

november

Enter a string

august

Output

string august is not found in the arraylist

Example - 3
Input

Enter size of an arraylist

4

Enter elements of an arraylist

high

medium

low

cheers

Enter a string

cheers

Output

string cheers is found in the arraylist
