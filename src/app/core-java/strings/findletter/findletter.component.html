Find a Letter in String


Program to Find whether the given key is present in a string or not
Example :
If we have a String "operations" and the key is 'r', then 'r' is present in 4th position of the "operations" then output should be "r is present in operations".

Sample 1 :
string = "hello java"
key = 'o'
output : o is present in hello java

Sample 2 :
string = "How are you"
key = 'j'
output : j is not present in How are you
Example - 1
Input

String = {{ '{' }}"hello java"{{ '}' }}

Key = 'o'

Output

o - present in the string - hello java

Example - 2
Input

String = {{ '{' }}"How are you"{{ '}' }}

Key = 'y'

Output

y - present in the string - How are you

Example - 3
Input

String = {{ '{' }}"May Month"{{ '}' }}

Key = 's'

Output

s - not present in the string - May Month
