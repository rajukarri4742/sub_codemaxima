Swap First and Last Letters of the String


Program to Swap First and last letters of a given String.
Example :
If we have String "hello" the first letter of the string "hello" is 'h' and the last letter of the string "hello" is 'o' , then 'h' will replace with 'o' and 'o' will replace with 'h' ,then output should be "String hello after swapping first and last letters is oellh".

Sample 1 :
String = "world"
Output : String world after swapping first and last letters is dorlw

Sample 2 :
String = "program"
Output : String program after swapping first and last letters is mrograp
Example - 1
Input

String = {{ '{' }}"world"{{ '}' }}

Output

dorlw

Example - 2
Input

String = {{ '{' }}"program"{{ '}' }}

Output

mrograp

Example - 3
Input

String = {{ '{' }}"madam"{{ '}' }}

Output

madam
