Spy Number

A number is said to be a Spy number if the sum of all the digits is equal to the product of all digits.

For Example:
Consider a number, 22
sum = 2 + 2 = 4
product = 2 * 2 = 4
Here, sum of all the digits of a number 22 and product of all the digits of a number 22 is equal.
So, 22 is a spy number.
Consider a number, 34
Here, sum of all the digits of a number 34 and product of all the digits of a number 34 is not equal.
So, 34 is not a spy number.

Example - 1
Input

Enter a number

22

Output

Given number 22 is a spy number

Example - 2
Input

Enter a number

123

Output

Given number 123 is a spy number

Example - 3
Input

Enter a number

1241

Output

Given number 1241 is not a spy number
