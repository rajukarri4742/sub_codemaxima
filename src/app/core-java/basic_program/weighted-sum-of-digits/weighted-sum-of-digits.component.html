Weighted Sum of Digits

Compute the weighted sum of digits of a number read from the keyboard in the following sense:
Let us calculate sum of digits multiplying each digit by its position (counting from the left, starting from 1).

For example
consider a value 1971.
we need to calculate such weighted sum of digits of that value as below
(1971) = 1 * 1 + 9 * 2 + 7 * 3 + 1 * 4 = 44

Example - 1
Input

1971

Output

44

Example - 2
Input

45

Output

14

Example - 3
Input

333

Output

18
