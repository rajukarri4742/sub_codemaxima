Palindrome

To print whether the given number is a palindrome or not. If the reverse of a number is equal to its original number then it is a palindrome number.

Example 1:
If the number is 4334 then the reverse of the number is 4334 which is equal to its original number.
So, 4334 is a palindrome number.
Example 2:
If the number is 2351 then the reverse of the number is 1532 which is not equal to its original number.
So, 2351 is not a palindrome number.

Example - 1
Input

Enter a number

4334

Output

Given number 4334 is a palindrome

Example - 2
Input

Enter a number

2351

Output

Given number 2351 is not a palindrome

Example - 3
Input

Enter a number

18881

Output

Given number 18881 is a palindrome
