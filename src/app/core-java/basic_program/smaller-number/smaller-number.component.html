Smaller Number

The program is to print the smaller number of given two numbers.

Example 1:
If the given two numbers are the 60 and 22
Program has to print 22 is lesser than 60.

Example 2:
If the given two numbers are the 11 and 85
Program has to print 11 is lesser than 85

Example - 1
Input

Enter First number

60

Enter Second number

22

Output

22 is lesser than 60

Example - 2
Input

Enter First number

11

Enter Second number

85

Output

11 is lesser than 85

Example - 3
Input

Enter First number

35

Enter Second number

90

Output

35 is lesser than 90
