Even or Odd

A number which is divisible by 2 and generates a remainder of 0 is called an even number.
A number that is not divisible by 2 and generates a remainder of 1 is called an odd number.

Example - 1
Input

Enter a number

22

Output

22 is even number.

Example - 2
Input

Enter a number

11

Output

11 is odd number.

Example - 3
Input

Enter a number

85

Output

85 is odd number.
