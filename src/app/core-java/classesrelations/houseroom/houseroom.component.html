House Class

Create house class with following properties
Create a tester class and create & print atleast 5 house objects in the tester class main method.

  • Declare all class variables as private.
  • Write Defalut and parameterized constructor.
  • Write Setters and Getters for each private variable.
  • Write toString() method for the class.

Here we are going to use room class object in the house class as a parameter.

Each house can have multiple rooms.

  1. doorno - The door number of the house.(string type)
  2. stairs - The number of floors in the house.(int type)
  3. streetname - The landmark of the house(String type)
  4. rooms - The list of rooms(List<Room> rooms - list object with Room type)
  5. square feet - The total area covered.(double type)
