Unchecked Exception


Try:
It is a block where an exception code present. we can't use try block alone and it must be followed by either catch or finally.

Catch:
Catch block will handle the exception. It should be preceded by try block. we can't use catch block alone without try block.

Throw:
It is used to execute the necessary code of the program. It is executed whether an exception is handled or not.

Throws:
It is used to throw an exception.

Finally:
It is used to declare exceptions. It will specify that there may occur an exception in the method. It doesn't throw an exception. It is always used with method signature.
