Constructor
  • Constructor is a special king of methos which name same as class name and doesn't have any return type.
  • The constructor is called when an object of a class is created with help of new keyword.
  • The defalut constructor automatically provided by the Java Object class(defalut parent of all classes of java).
  • Constructor is also used to set the class properties through parameter constructor.
  • If we write parameter constructor in the class you have to provide defalut constructor as well other wise we not able to create defalut(empty) object.

Getters & Setters
  • Getters and Setters are the public methods with used to bind the private data of the class.
  • Getter and Setter are methods used to secure your data by making your class data as private(only access in the class).
  • Getters are used to access the data. Setter are used to set the data.
  • For each class variable we will write seprare getter and setter methods.
  • The advantage of binding private variables using public methods, we can add some validation operations for the data.
