Reverse of an Array

In this program, we need to print the elements of the array in reverse order that is,
the last element should be displayed first, followed by second last element and so on.

for example: 

1  2  4  5   7

Above array in reversed order:

7  5  4  2   1

Method 1:

we can print the array from last to first i.e consider below

for (int i = arr.length-1; i >= 0; i--)  
      System.out.print(arr[i] + " ");  


Method 2:

By using temporary array also, we can reverse the array. Consider the below,

for example:

int size=sc.nextInt();

int a[]=new int[size];
int b[]=new int[size];

we need to print the array "a" in the reverse order. So, we are considering temporary array i.e array "b".
So,we can take the last element in the array "a" and fill it in the first position in the second array "b".
In the same way, the second last element in the array "a" should be displayed in the second element in the array "b" and so on...
Example - 1
Input

Array = {{ '{' }}11, 22, 4, 85, 7{{ '}' }}

Output

Reverse of an array is: 7 85 4 22 11

Example - 2
Input

Array = {{ '{' }}11, 13, 45, 22, 65{{ '}' }}

Output

Reverse of an array is: 65 22 45 13 11

Example - 3
Input

Array = {{ '{' }}24, 22, 56, 22, 22{{ '}' }}

Output

Reverse of an array is: 22 22 56 22 24
