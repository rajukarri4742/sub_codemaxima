Super Array or Not

Read the inputs (mention the size of the inputs) and take any random number.
if the addition of any two numbers in the array is equal to the random number then print the array is super array or not a super array.

INPUT:
size -- 5
enter random number -- 14
values in the array
2 3 6 8 9
here if we add any two numbers then the answer should be equal to the random number 14 then print super array else not a super array.
2+3 = 5   --false
2+6 = 8   --false
2+8 = 10  --false
2+9 = 11  --false
3+6 = 9   --false
3+8 = 11  --false
3+9 = 12  --false
6+8 = 14  --true
break the loop and print it is a super array.

OUTPUT:
SUPER ARRAY
