Number of Pairs in Array

Print the number of pairs (should be the same number) can form in an array.

INPUT:
size -- 6
enter values:
20 10 20 10 10 40

OUTPUT:
first pair-- 20,20
second pair -- 10,10
single- 10 and single 40 ---so these numbers are not formed as a pair
So, 2 pairs are formed in the array.
Example - 1
Input

Array = {{ '{' }}2, 4, 2, 5, 6, 4, 4{{ '}' }}

Output

The number of pairs are : 2

Example - 2
Input

Array = {{ '{' }}15, 22, 15, 29, 22, 29{{ '}' }}

Output

The number of pairs are : 3

Example - 3
Input

Array = {{ '{' }}71, 71, 22, 22, 22, 22, 22, 85{{ '}' }}

Output

The number of pairs are : 3
