Reverse Each Value in Array

We can iterate every element in the array and then reverse the number and replace the reversed number in the same position. 
Consider the below for better understanding.

int a[] = {{ '{' }}12,45,67{{ '}' }}

The output will be as below
{{ '{' }}21,54,76{{ '}' }}

Step 1: we can take the first number 12 and need to reverse the number and replace the number in the same place.
        so a[0] = 12
        after reverse the number.
        a[0] = 21

Step 2: we can take the next number 45 and need to reverse the number and replace the number in the same place.
        so a[1] = 45
        after reverse the number.
        a[1] = 54

Step 3: we can take the next number 67 and need to reverse the number and replace the number in the same place.
        so a[2] = 67
        after reverse the number.
        a[2] = 76.
Example - 1
Input

Array = {{ '{' }}1, 6, 14, 85, 17{{ '}' }}

Output

The reverse of each element in the array is 1 6 41 58 71

Example - 2
Input

Array = {{ '{' }}15, 23, 45, 29, 43{{ '}' }}

Output

The reverse of each element in the array is 51 32 54 92 34

Example - 3
Input

Array = {{ '{' }}71, 73, 26, 89, 97{{ '}' }}

Output

The reverse of each element in the array is 17 37 62 98 79
