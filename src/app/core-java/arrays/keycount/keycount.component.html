Find Key(Element) Count in Array

This example is for how many times the number found in an array
Consider the below steps.
                      
int arr[] = {{ '{' }}1,3,5,5,7{{ '}' }}
key = 5 
We are going to find how many times 5 present in the given array.
                      
Step 1: first we need to compare the number 5 with the first element in the array i.e 1
                      
        ----> 5 == 1 (false) 
                      
Step 2: then move to next number in the array i.e 3
                      
        ----> 5 == 3 (false) 
                      
Step 3: again move to next number i.e 5
                      
        ----> 5 == 5 (true)

       we can take a varaible count;
       if we found the number 5 in an array then increase the count i.e count+1= 0+1 = 1
       S0, count =1;
                      
Step 4: again move to next number i.e 5
                      
        ----> 5 == 5 (true)
        Again we found the number 5 in an array then again increase the count, previously count is 1 and again we need to increarse the count i.e 2
        S0, count =2;
                      
Step 5: then move to next number in the array i.e 7
                      
        ----> 5 == 7 (false) 
                      
        Output: 2 times found the number in an array(count=2)
Example - 1
Input

Array = {{ '{' }}1, 3, 5, 5, 7{{ '}' }}

Key = 5

Output

2 times 5 found in an array

Example - 2
Input

Array = {{ '{' }}11, 13, 85, 22, 65{{ '}' }}

Key = 85

Output

0 times 85 found in an array

Example - 3
Input

Array = {{ '{' }}24, 22, 85, 22, 22{{ '}' }}
