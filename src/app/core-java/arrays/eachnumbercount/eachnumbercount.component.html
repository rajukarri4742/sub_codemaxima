Each Number Count in Array

We need to print the number of occurrences of each number in the array.

INPUT:
size -- 6
enter values 
2 4 2 5 6 4

OUTPUT:
2 occurs two times
4 occurs two times
5 occurs one times
6 occurs one times
Example - 1
Input

Array = {{ '{' }}2, 4, 2, 5, 6, 4, 4{{ '}' }}

Output

2 occurs 2 times

4 occurs 3 times

5 occurs 1 time

6 occurs 1 time

Example - 2
Input

Array = {{ '{' }}15, 22, 15, 29, 22, 29{{ '}' }}

Output

15 occurs 2 times

22 occurs 2 times

29 occurs 2 times

Example - 3
Input

Array = {{ '{' }}71, 71, 22, 22, 22, 22, 22, 85{{ '}' }}

Output

71 occurs 2 times

22 occurs 5 times

85 occurs 1 time
