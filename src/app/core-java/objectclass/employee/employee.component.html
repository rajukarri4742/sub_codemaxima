Employee Class

Create Employee class with following properties
Create a tester class and create & print atleast 5 Employee objects in the tester class main method.

  1. empid - This property is to give unique id for the employee like employeeID(string type)
  2. name - The name of the employee.(string type)
  3. designation - The role of the employee like(software engineer, associate consultant, backend developer and etc)(String type)
  4. salary - The package of the employee.(double type)
  5. experience - The total years of the employee obtainded.(double type)
  6. permanent - The status of the employee whether the employee is permanent or not(true or false).(boolean type)
