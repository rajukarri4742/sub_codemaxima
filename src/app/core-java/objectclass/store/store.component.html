Create Store class with following properties
Create a tester class and create & print atleast 5 Store objects in the tester class main method.

  1. id - This property is to give unique id for the store like storenumber(int type)
  2. name - The name of the store.(string type)
  3. cards payment - The payments done for the store either by cash or card(boolean type)
  4. store value - The value of the store.(double type)
  5. store type - The type of the store like supermarket, clothing store, vegetable market.(string type)
