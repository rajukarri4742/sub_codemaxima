Introduction of Methods

Mehtods
A method is a block of code or collection of statements or a set of code grouped together to perform a certain task or operation.
A method is a group of statements that perform some operation on some data, and may or may not return a result.
Method Name
It is used to give a unique name to the method. In the above syntax, “add” is the method name.
Method Signature/Parameters.
It is the list of parameters separated by a comma and enclosed in the pair of parentheses. 
It contains the data type and variable name. If the method has no parameter, left the parentheses blank.
Return Type 
It defines the return type of the method. In the above syntax, “int” is the return type.
So, it should return some integer value.
