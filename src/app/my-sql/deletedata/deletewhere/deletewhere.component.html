DELETE with Where

The DELETE statement is used to delete existing records in a table. You can use the WHERE clause with a DELETE query to delete the selected rows, otherwise all the records would be deleted. 


                      Syntax : DELETE FROM table_name WHERE condition;
                                      
                      E.g.: - DELETE FROM CUSTOMERS where id = 3;

                    QUESION : DELETE a row of customer table where id is 3.

                    
        DELETE FROM CUSTOMERS where id = 3;
