DELETE Statement

The DELETE statement is used to delete existing records in a table. If you want to DELETE all the records from the CUSTOMERS table, you do not need to use the WHERE clause and the DELETE query. 


                      Syntax : DELETE FROM table_name;
                                      
                      E.g.: - DELETE FROM CUSTOMERS;

                    QUESION : DELETE the all rows of customer table.

                    
        DELETE FROM CUSTOMERS;
