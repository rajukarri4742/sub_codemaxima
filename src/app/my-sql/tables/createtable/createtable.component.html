CREATE TABLE Statement

Tables are used to store data in the database. Tables are uniquely named within a database and schema. Each table contains one or more columns. And each column has an associated data type that defines the kind of data it can store. 


        Syntax : CREATE TABLE table_name(
            column1 datatype,
            column2 datatype,
            column3 datatype,
            .....
            columnN datatype
         );
                                      
        E.g.: - CREATE TABLE Persons (
            PersonID int,
            LastName varchar(255),
            FirstName varchar(255),
            Address varchar(255),
            City varchar(255)
        );
